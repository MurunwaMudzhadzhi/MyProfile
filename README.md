# Murunwa Mudzhadzhi – Developer Portfolio

## How to Navigate This Project

### 1. Open the Project
Locate the file: `index.html`
Open it in any web browser (Google Chrome recommended)

### 2. Landing Section (Hero)
Displays name, role, and introduction
Use buttons:
View Projects → scrolls to Projects section
Get In Touch → scrolls to Contact section

### 3. Navigation Bar
Fixed at the top of the page
Click to jump to:
Skills
Projects
Education
Contact


### 4. Scrolling
Scroll down to explore the portfolio
Sections animate into view automatically


### 5. Skills Section
Shows technical skills grouped into:
Languages
Databases
Web
Tools
Practices



### 6. Projects Section
Displays completed projects
Each project includes:
Title
Description
Technologies used


### 7. Education SectionShows academic background and qualifications


### 8. Competencies Section
Lists key strengths and abilities


### 9. Contact Section
Includes:
Email (clickable)
Phone numbers
Location
LinkedIn


### 10. Background & Animations
Animated background and robot run automatically
No user interaction required


### 11. Mobile View
Layout adjusts automatically for smaller screens
Navigation is simplified

## End of Guide
